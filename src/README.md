# アプリケーション名

first-laravel

## 環境構築

-   docker-compose up -d --build
-   docker-compose exec php bash
-   composer create-project "laravel/laravel=8.\*" . --prefer-dist
-   php artisan migrate

## 使用技術(実行環境)

-   Laravel8.6.12
-   php7.4.9
